# متجرعات المستقبل - بوت تيليجرام تعليمي

بوت تفاعلي لحساب العد التنازلي لامتحانات الثانوية العامة، وتحفيز الطلاب بآيات قرآنية وأزرار تفاعلية.

## الأوامر:
- `/start` - عرض لوحة التفاعل.
- `/countdown` - حساب الأيام والساعات المتبقية للامتحان.

## التشغيل على Railway:

1. اربط هذا المشروع من GitHub.
2. ضيف متغير بيئة `BOT_TOKEN` في صفحة Settings.
3. Railway راح يشغّل البوت تلقائياً.

تم التطوير بأسلوب مستقبلي لتحفيز الطلاب.
